# rhio

> MINIO + HIRO + IROH = RHIO

## Usage

```
p2p blob syncing node for minio databases

Usage: rhio [OPTIONS]

Options:
  -p, --bind-port <PORT>                           
  -t, --direct-node-addresses [<TICKET>...]        
  -k, --private-key <PATH>                         
  -d, --sync-dir <PATH>                            
  -i, --import-path <PATH | URL>                   
  -c, --minio-credentials <ACCESS_KEY:SECRET_KEY>  
  -r, --relay <URL>                                
  -h, --help                                       Print help
  -V, --version                                    Print version
```
